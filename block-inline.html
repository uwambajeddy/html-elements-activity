Block vs Inline Elements Demo

This is a paragraph with some text. Here's a strong word which is an inline element. And here's a highlighted span with a different background color.

This is a div element (block-level). Notice how it takes up the full width and creates a new line.
This is a section element (also block-level). It also takes up the full width and stacks below the div.

Notice how the block elements stack vertically and take up the full width, while the inline elements (strong and span) flow within the text without breaking the line.
