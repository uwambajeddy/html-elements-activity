About iFrames

An iframe (inline frame) is used to embed another HTML document within the current document. Below you can see our child.html page embedded inside this parent page.

The iframe creates a separate browsing context, so the embedded page maintains its own styles and scripts independently from the parent page.

Embedded Child Page

Key Points about this iFrame:

  • Width: 650 pixels
  • Height: 500 pixels
  • Border: 3px solid black with rounded corners
  • Source: child.html (in the same folder)
  • Accessibility: Includes a title attribute
  • Fallback: Provides alternative content for browsers that don't support iframes
